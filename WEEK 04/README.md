# E-commerce Product Landing Page

A fast, responsive, and visually stunning E-commerce Product Landing Page built with pure vanilla technologies (HTML, CSS, JS). Features a premium dark theme, dynamic data generation, and simple cart integration.

## Project Structure

```
WEEK 04/
├── index.html        # Main landing page with hero & products
├── about.html        # About us details
├── services.html     # Service offerings grid
├── contact.html      # Responsive contact form
├── css/
│   └── style.css     # 100% Custom modern CSS (no framework)
├── js/
│   └── script.js     # Vanilla JS powering dynamic UI
├── images/           # Local assets holder (if needed)
└── README.md         # Project documentation
```

## Core Features

- **No Frameworks:** Developed entirely using native web capabilities.
- **Dynamic Content Injection:** Using JavaScript to generate the product grid dynamically.
- **Glassmorphism & Premium UI:** Utilizing standard modern design patterns.
- **Local Storage State:** Keeps cart state alive across reloads natively.

## Getting Started

1. Simply clone this folder or open the directory.
2. Open `index.html` in your favorite modern browser (Chrome, Firefox, Safari, Edge).
3. Navigate across pages via the header.
