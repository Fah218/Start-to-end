# Interactive Portfolio Website

This portfolio was built in 3 stages:

## Week 1 - HTML
- Semantic structure
- About, Skills, Contact sections
- Contact form
- Internal navigation

## Week 2 - CSS
- External stylesheet
- Flexbox layout
- Responsive design
- Hover effects
- Custom Google Font

## Week 3 - JavaScript
- Form validation with error messages
- Dark mode toggle
- Image zoom effect
- Dynamic footer year
- DOM manipulation & event listeners
- LocalStorage support

## Run Project
Open index.html in your browser.
